# Form Validation App ✅

A simple form validation project that checks user input and provides real-time feedback.

## 🚀 Features
- Validate required fields
- Check email format
- Password length validation
- Error and success messages
- User-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Project Goal
This project was built to practice JavaScript form validation and improve user input handling skills.
